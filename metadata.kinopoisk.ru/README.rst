Kinopoisk.ru movie scraper for XBMC
===================================

Description
-----------

Download Movie information from www.kinopoisk.ru

Supported tags::

  title
  originaltitle
  country
  year
  top250
  mpaa
  tagline
  runtime
  rating
  genre
  studio
  actor
  plot and outline (same content)
  id
  thumb
  actor
  director
  credits
  producer
  fanart (from themoviedb.org)
  trailer (only russian language)
